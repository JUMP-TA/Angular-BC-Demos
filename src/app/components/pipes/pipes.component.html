Casing

Transform the casing of text.

Original: {{ title }}

Lowercase: {{ title | lowercase }}

Uppercase: {{ title | uppercase }}

Titlecase: {{ title | titlecase }}

Slice

Slices / cuts a string.

Original: {{ title }}

Starting at index 3: {{ title | slice:3 }}

Starting at index 3, up to index 7: {{ title | slice:3:7 }}

JSON

Allows you to print a string representation of JSON data.

Original: {{ movie }}

JSON: {{ movie | json }}

Numbers

Allows you to manipulate how a number is displayed.

Original: {{ num }}

'1.2-3' = {{ num | number:'1.2-3' }}

'3.4-5' = {{ num | number:'3.4-5' }}

'3.1-2' = {{ num | number:'3.1-2' }}


Original: {{ percNum }}

Percent: {{ percNum | percent }}

Percent: {{ percNum | percent:'1.2' }}

Currency

Allows to format a number so it displays as a currency.

Original: {{ currVal }}

Currency: {{ currVal | currency }}

Currency in British Pounds (GBP): {{ currVal | currency: 'GBP' }}

Currency in British Pounds With Code: {{ currVal | currency: 'GBP': 'code' }}

Dates

Can format a date.

Original: {{ date }}

Date: {{ date | date }}

Short: {{ date | date: 'short' }}

Short Date: {{ date | date: 'shortDate' }}

Short Time: {{ date | date: 'shortTime' }}

Custom: {{ date | date: 'yyyy-MM-dd hh:mm:ss' }}

Custom Pipe: Double

Doubles the string given to it.

{{ 'hello' | double }}

{{ 5 | double }}
